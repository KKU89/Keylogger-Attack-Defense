Keylogger Attack & Defense Project:-

This Python project demonstrates how a keylogger works and how to detect it using process monitoring. Made for cybersecurity learning only.

Requirements:-
- Python 3.x
- pynput
- psutil

Installation:-
pip install pynput
pip install psutil

How to run:-

Run Keylogger:
cd keylogger
python keylogger.py

Run Detector:
cd detector
python keylogger_detector.py
